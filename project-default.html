PROJECT CASE STUDY

Commerce Experience Revamp

이탈률이 높은 구매 여정 구간을 개선하여 전환율과 체류 시간을 함께 상승시킨 리뉴얼 프로젝트입니다.

기간

12주

역할

Frontend Lead

성과

전환율 +18%

문제 정의

초기 분석에서 핵심 이탈 구간은 제품 상세-장바구니-결제 단계로 확인되었고, 페이지 응답속도와 정보 복잡도가 주요 원인이었습니다.

해결 전략

  • • 여정 단계별 핵심 의사결정 정보 우선 배치
  • • 이미지 최적화 및 코드 스플리팅으로 LCP 개선
  • • A/B 테스트로 UI 변경의 실제 성과 검증
